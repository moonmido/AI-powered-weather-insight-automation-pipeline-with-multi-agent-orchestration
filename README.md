# 🌍 AI Weather Voice Agent System

**An AI-powered automation workflow** that transforms a simple user input (**city + email**) into a fully generated **spoken weather report (MP3)** delivered via email with Google Drive storage.

This system combines **AI agents**, real-time weather APIs, and automation pipelines to deliver intelligent, human-like weather insights in natural audio format.

---

## 🚀 Core Idea

**User provides:**
- City name 🌍
- Email address 📧

**System delivers:**
- ✅ AI-corrected city name
- 🌡️ Real-time weather data
- 🤖 Intelligent AI-generated weather explanation
- 🎵 High-quality MP3 audio (ElevenLabs TTS)
- ☁️ Google Drive storage link
- 📧 Email notification with direct access
